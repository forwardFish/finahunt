# Epic 2A.2 Theme Cluster and Candidate Mapping

这个 Epic 负责把分散事件真正组织成可研究的题材对象，并映射出“最值得跟踪的票”。

重点不是给更多新闻，而是回答：

- 这些信息是否属于同一个更高层题材
- 哪些标的是最正宗、最稀缺、最值得跟踪的候选
- 哪些只是擦边、跟风或需要过滤的对象
