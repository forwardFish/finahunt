工信部：加快推进人形机器人创新发展 培育未来产业新赛道

产业工信部2025-05-20 09:48来源：工信部官网作者：机器人产业处☆ 收藏
所属题材： 人形机器人 AI 应用 智能制造

工信部印发《人形机器人创新发展指导意见》，提出到2027年形成安全可靠的产业体系，推动人形机器人在制造业、服务业、家庭等场景的规模化应用。

意见从关键技术攻关、整机产品突破、应用场景拓展、产业生态构建等方面提出系统性部署，加快推动人形机器人产业高质量发展。

核心要点

  • 到2027年，形成安全可靠的产业体系，培育2-3家具有全球影响力的生态型企业；
  • 突破运动控制、感知交互、人工智能等关键技术，提升整机产品可靠性；
  • 拓展制造业、服务业、家庭等多场景应用，打造标杆示范工程；
  • 完善标准体系与测试验证平台，推动产业协同创新与开放合作。
