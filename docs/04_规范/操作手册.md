# 操作手册

## 本地初始化

1. 创建 Python 3.11+ 环境。
2. 执行 `pip install -e .[dev]`。
3. 运行 `pytest`。

## 运行工作流

1. `python -c "from workflows.build_workflow import run_build_workflow; print(run_build_workflow('init'))"`
2. `python -c "from workflows.runtime_schedule import run_runtime_cycle; print(run_runtime_cycle())"`

## 审批策略

当前默认策略为自动通过，位于 `context.auto_approve=True`，与当前执行要求一致。
